## 游米链YMC智能合约

说明：本智能合约使用truffle开发框架。

## 目录
* [功能列表](#功能列表)
* [安装说明](#安装说明)


<h2 id="功能列表">功能列表</h2>

- 支持ERC20标准的所有功能
- 管理游戏种类（新增、修改、删除）
- 管理平台用户（新增、修改、删除）
- 管理平台用户所属的多个游戏账号（新增、修改、删除）
- 管理游戏日志（新增）


<h2 id="安装说明">安装说明</h2>

#### 1. 安装依赖库

```
npm install openzeppelin-solidity
```
#### 2. 编译

```
truffle compile
```
